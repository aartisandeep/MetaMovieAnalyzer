**MetaMovieAnalyzer**

MetaMovieAnalyzer is a Python application designed to extract and analyze movie data from Metacritic, focusing on directors and main casts of the top 1000 movies.
It employs BeautifulSoup and requests for web scraping, processes and structures the data, and offers insightful analysis including director comparisons using cosine similarity.

**Features**
**Web Scraping:** Uses BeautifulSoup and requests to scrape data for the top 1000 movies from Metacritic.
**Data Processing:** Builds a detailed list of movies, including directors and main casts.
**CSV Data Storage:** Stores extracted data in new_movies.csv for easy access and analysis.
**Cosine Similarity Computation:** Includes a feature to compare directors based on actor collaborations.
